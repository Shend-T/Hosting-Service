### Struktura e Folder-ave

`/vendor` - Ktu ndodhen 'external PHP libraries'
`/tests` - Ktu testohen `API endpoints` permes file-it `phpunit.xml`
`/storage` - Ktu ruhet dynamic data
`/storage/public` - File-at qe user-i mund tju qaset permes URL
Ktu psh mund te ruhen pdf file-at qe user-i mund ti download-on
`/storage/private` - File-at qe user-i nuk mund tju qaset
